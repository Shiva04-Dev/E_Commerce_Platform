# E-Commerce Platform

A full-stack e-commerce application built with Spring Boot and PostgreSQL to provide a complete online shopping experience for customers and administrators.

Customers can browse products, manage their shopping cart, and complete purchases, while administrators can manage inventory, track orders, and oversee the entire platform; all within a secure, role-based system.
